# Eva
Eva is name pronunciation cloud based AI, which leverages Azure platform to pronunce any given name based on locale accent.

## Installation
### Windows
```sh
pip install -r requirements.txt
```

### Unix
```sh
pip3 install -r requirements.txt
```
